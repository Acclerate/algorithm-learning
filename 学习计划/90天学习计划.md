## 🎯 每日学习流程（60 分钟）

```bash
# ┌─────────────────────────────────────┐
# │ 00:00 - 00:10  复习昨天内容         │
# │   - 只看代码，口述思路              │
# │   - 卡住则标记"需重做"              │
# ├─────────────────────────────────────┤
# │ 00:10 - 00:15  读新题，思考         │
# │   - 想不出来立刻看题解              │
# │   - 不要死磕超过 5 分钟             │
# ├─────────────────────────────────────┤
# │ 00:15 - 00:30  理解题解             │
# │   - 推荐看视频或动图                │
# │   - 重点理解"为什么用这个数据结构"  │
# ├─────────────────────────────────────┤
# │ 00:30 - 00:40  照抄代码             │
# │   - 边抄边理解每一行的逻辑          │
# │   - 手写效果最好                    │
# ├─────────────────────────────────────┤
# │ 00:40 - 01:00  凭记忆重写           │
# │   - 关掉所有参考                    │
# │   - 15 分钟内完成 = 合格            │
# │   - 否则明天必须复习                │
# └─────────────────────────────────────┘
```

## ⚠️ 重要提醒

1. **不要追求做完所有题目**
   - 150 题里约有 30-40 道 Hard
   - 第一遍直接**跳过 Hard**
   - 掌握 70-80 道 Easy/Medium 足矣

2. **质量 > 数量**
   - 一道题做 3 遍 > 3 道题各做 1 遍
   - 白板测试不通过 = 没学会

3. **模式记忆**
   - 不要背题目，要背**模式**
   - 每个模式至少做 3 道题巩固

4. **定期复习**
   - 1 天后复习
   - 3 天后复习
   - 1 周后复习
   - 1 个月后复习
