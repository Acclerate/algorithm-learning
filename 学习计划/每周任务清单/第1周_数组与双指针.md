## 💡 本周学习要点

### 为什么学双指针？

#### 问题：嵌套循环的痛点
```java
// ❌ 暴力解法 - O(n²)
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        // 嵌套循环，时间复杂度高
    }
}
```

#### 解决：双指针优化 - O(n)
```java
// ✅ 双指针 - O(n)
int i = 0, j = n - 1;
while (i < j) {
    // 两个指针协同移动，避免嵌套循环
}
```

### 双指针的 4 种模式

#### 1️⃣ 从后往前的双指针
- **应用**: LeetCode 88 合并两个有序数组
- **关键**: 避免覆盖已有数据
- **方向**: 从数组末尾开始向前移动

#### 2️⃣ 快慢指针
- **应用**: LeetCode 27 移除元素、LeetCode 26 删除重复项
- **关键**: 快指针探索，慢指针标记有效位置
- **方向**: 都从前往后，但速度不同

#### 3️⃣ 左右相向双指针
- **应用**: LeetCode 167 两数之和 II、LeetCode 11 盛水容器
- **关键**: 一个从左，一个从右，向中间靠拢
- **方向**: 从两端向中间

#### 4️⃣ 贪心 + 双指针
- **应用**: LeetCode 121 买卖股票、LeetCode 55 跳跃游戏
- **关键**: 每次做局部最优选择
- **思想**: 双指针 + 贪心策略
